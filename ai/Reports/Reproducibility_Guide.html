2. Input Data and File Structure

The following files must be located in the same directory as the notebook/scripts:

File Name Content Importance
engineered_stock_data.csv All stock data with engineered features and the target column (Return_7d). Essential
catboost_ranker_optimized.cbm The final trained CatBoost Ranker model. Essential for Backtest
scaler.pkl The $\text{StandardScaler}$ object **fitted on the training data**. Essential for consistent data scaling.
Note: All backtest results rely on the existence of these files and the exact time-based data split performed in the code.

3. Critical Parameters for Replication

A) Optimized CatBoost Model Parameters

These hyperparameters must be used during model training to ensure exact result replication (random_seed=42 is crucial):

Parameter Value Role in Reproducibility
loss_function YetiRank The listwise ranking objective.
iterations 3000 Total number of allowed trees.
depth 7 Maximum tree depth.
random_seed 42 Guarantees deterministic training process.
early_stopping_rounds 150 Prevents overfitting based on validation set.

B) Operational Backtest Parameters

Parameter Value Operational Context
Optimal Holding Period 5 Days Configuration yielding the highest Sharpe Ratio.
Top K (Recommendations) 5 Number of stocks purchased daily.
Slippage Rate 0.0005 (0.05%) Fixed slippage applied to each trade.
Liquidity Filter Volume > 100K Applied during the backtest simulation.
